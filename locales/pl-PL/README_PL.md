# Indeks Masy Ciała (BMI) Kalkulator - Kalorie

1- Wprowadź wagę
2- Wprowadź wzrost
3- Index masy ciała = waga / wzrost²

# Wskaźnik Metaboliczny Basala (BMR) Kalkulator

Wprowadź poniższe wartości:

1 - Wiek
1 - Waga
2 - Wzrost
3 - Częstotliwość ćwiczeń

4 - Płeć (Mężczyzna/Kobieta)

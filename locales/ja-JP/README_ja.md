# ボディ・マス・指数計算機 (BMI) - カロリー

1- 体重を入力します。
2- 身長を入力します。
3- BMI = 体重/身長²

# 基礎代謝率 (BMR) 算機 

次の情報を入力してください:

1 - 年齢 
1 - 体重 
2 - 身長 
3 - 運動率 
4 - 性別(男性/女性)
